# FithubFast

Actix Web（Rust）バックエンドと、React（Vite）フロントエンドで構成されたフィットネス系アプリのプロジェクトです。

## 主な機能

### 📊 ダッシュボード
- **アクティビティヒートマップ**: 年間のトレーニング記録をGitHubライクなヒートマップで可視化
- **統計表示**: 週のワークアウト回数、総重量、トレーニング継続日数、ログイン継続日数
- **レベル＆経験値**: トレーニングの成長指標として可視化
- **ストリークボーナス**: トレーニング継続やログイン継続でEXP倍率アップ
- **筋肉回復ステータス**: 各部位の回復状況（回復中/トレーニング可能/休み過ぎ）を確認
- **直近のトレーニング履歴**: 最近のワークアウト記録をクイック表示

### 📝 トレーニング記録
- **日別記録管理**: カレンダーから日付を選択してトレーニングを記録
- **セット管理**: 種目ごとに重量・回数・セット数を記録
- **PR（自己記録）追跡**: 各種目の最高記録を自動判定・表示
- **カスタム種目**: 自分だけのオリジナル種目を追加可能
- **タグ管理**: 種目にタグを付けて整理

### 🏋️ エクササイズ図鑑
- **種目検索**: 部位・難易度・ターゲット筋で絞り込み
- **詳細情報**: 各種目のフォーム説明、効く筋肉、難易度を確認
- **無限スクロール**: スムーズなブラウジング体験

### 💊 サプリメントガイド
- **カテゴリ別一覧**: アミノ酸系、プロテイン系、ビタミン・ミネラル系、パフォーマンス系
- **ティア評価**: 各サプリメントをS〜Cでランク付け（S=効果が高い、A=効果あり、B=補助的に有効、C=限定的な効果）
- **総合ティア表示**: 全カテゴリを横断してティア順に表示
- **詳細モーダル**: 効果、摂取タイミング、注意点などを確認

### 🎒 トレーニングギアガイド
- **カテゴリ別紹介**: 各種ギアの特徴と選び方
- **タイプ比較**: カテゴリ内の各タイプを詳細比較
- **初心者向けヒント**: 段階的なギア選びのアドバイス

### 🗺️ ジム検索（仙台市）
- **地域フィルタ**: 青葉区、宮城野区、若林区、太白区、泉区
- **条件検索**: キーワード、料金上限、設備タグで絞り込み
- **マップ表示**: 地図上でジムの位置を確認
- **無限スクロール**: 多数のジムをスムーズに閲覧

### 🐾 パートナー（ペット）システム
- **パートナー育成**: トレーニングを頑張るとペットが成長
- **ムードシステム**: トレーニング頻度でペットの気分が変化
- **ステージ進化**: 卵からスタートして段階的に成長

### ⚙️ 設定・その他
- **プロフィール設定**: 表示名、性別、身長、体重などを管理
- **OAuth認証**: Google、GitHub、Microsoft でログイン

## 構成

- バックエンド: Rust / Actix Web
- フロントエンド: React / Vite
- 静的配信: `static/`
- ドキュメント: `docs/`

## ディレクトリ構成（主要）

- `src/`: Rustバックエンド
	- `api/`: APIエンドポイント
	- `auth/`: 認証
	- `db/`: DBモデル/接続
	- `middleware/`: ミドルウェア
- `frontend/`: Reactフロントエンド
	- `src/`: UI実装
	- `public/`: 公開アセット
- `static/`: ビルド済みフロントエンド資産
- `docs/`: 設計/運用ドキュメント
- `tests/`: 統合テスト

## 事前準備

- Rust（stable）
- Node.js（LTS推奨）
- npm（またはpnpm/yarn）

## 開発手順

### バックエンド起動

```
cargo run
```

### フロントエンド起動

```
cd frontend
npm install
npm run dev
```

## ビルド

### フロントエンド

```
cd frontend
npm run build
```

### バックエンド

```
cargo build --release
```

## テスト

```
cargo test
```
